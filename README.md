# ChessTimer
ChessTimer is a minimal, mobile-friendly chess clock for two players sharing one phone. Built as a progressive web app, it works instantly in any mobile browser with no download required — just open and play.
Features

8 preset time controls: 1, 2, 3, 5, 10, 15, 20, and 30 minutes
Flip clock display with animated tile transitions
Auto-pause when the app is backgrounded or the screen locks
Screen wake lock to prevent the display from dimming mid-game
Clean two-player layout with each clock facing its player
Installable as a PWA — works like a native app from your home screen

Built with HTML, CSS, and vanilla JavaScript. Fonts by Adobe Fonts (Futura PT). Designed in Figma.
